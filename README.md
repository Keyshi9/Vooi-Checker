# VOOI Airdrop Checker (Unofficial)

An unofficial airdrop estimator for VOOI, designed with the VOOI Pro aesthetic.

## Features
- **Real-time Estimation**: Calculate your potential allocation based on your points.
- **Multiple Scenarios**: View estimates for Bearish ($112.5M) to Bullish ($2.5B) FDV.
- **VOOI Design**: Dark mode, neon orange accents, and glassmorphism UI.
- **Responsive**: Works on mobile and desktop.

## How to Use
1. Clone the repository.
2. Open `index.html` in your browser.
3. Enter your points to see the results.

## Disclaimer
This is an unofficial tool and is not affiliated with VOOI. All calculations are estimates based on public information (420M supply, 7.5% allocation).
